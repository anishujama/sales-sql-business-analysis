# Sales SQL Business Analysis

## Overview

This project focuses on analyzing a sales transaction dataset using SQL to answer business-oriented questions and generate actionable insights.

## Objectives

- Analyze revenue and profit
- Identify top and bottom performing products
- Evaluate category-wise performance
- Analyze regional sales trends
- Study customer payment preferences
- Compare sales channels

## SQL Concepts Used

- SELECT
- SUM()
- AVG()
- COUNT()
- GROUP BY
- ORDER BY
- LIMIT
- Aggregation Functions

## Business Questions Solved

1. Total Revenue
2. Total Profit
3. Average Order Value
4. Top 5 Products by Revenue
5. Bottom 5 Products by Revenue
6. Revenue by Category
7. Profit by Category
8. Revenue by Region
9. Most Used Payment Method
10. Revenue by Sales Channel

## Tools Used

- MySQL
- SQL
- CSV Files
- GitHub

## Outcome

Generated business insights from transactional data using SQL aggregation and analytical queries.
